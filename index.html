    About

    A repo of commonly used tools and tips for dealing with data.

    Specifically geared around preparing data for further analysis and visualization.

    We will focus on using d3.js and lodash for many of these tasks.

    Code

    Each document in this repo is executed when loaded into a browser. Check it out by opening the Developer Tools Console. You should see the output of the following code block:

    console.log("This is the index!");
